For Sonar 5.2+

Number of seconds to wait for subsequent attempt to query SonarQube Compute Engine API to determine analysis results. SonarQube 5.2+ onwards performs final processing of reports asynchronously and exposes the results through an API called Compute Engine. The API needs to be polled to determine completion status. This field stores the time interval in seconds for subsequent GET call aattempts to the CE Api

Recommended value: 10
